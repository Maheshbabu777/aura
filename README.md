# AURA - Personal AI Operating System

A local-first, cloud-hybrid AI system for autonomous task management and goal tracking.

## Status

Week 0: Environment Setup

## Prerequisites

- Python 3.11+
- Ollama
- Google AI Studio API key

## Setup

```bash
# Pull local model
ollama pull gemma4:e4b

# Configure environment
cp .env.example .env
# Add your GEMINI_API_KEY to .env
```
